# Fullstack Chat App

### Description

- MongoDB created in Atlas
- Socket.io for the chatrooms

### Backend

#### start server: `node index.js`

- `npm install express`
- `npm install http`
- `npm install socket.io`
- `npm install mongodb`
- `npm install cors`
- `npm install dotenv`
- `npm i -D nodemon`

### Frontend

##### start server: `npm run dev`

- `npx create-next-app@latest chat-app-fs`
